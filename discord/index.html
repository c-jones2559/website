Discord Bot

I have a bot!

My Discord bot features over 20 commands from moderation to fun games.
My favourite commands are tictactoe and quote, but there's a full list of them below!
All messages sent to the bot are forwarded to my Discord account, so it's a great way to contact me!

Commands

Normal Commands

These are the commands usable by anyone.
/dm
Sends a message to someone for you after searching for them by id or username.
/dmuser
Sends a message to someone for you after picking them from a dropdown list.
/echo
Sends back the message after the command.
/help
Displays a help message.
/info
Sends details about the bot.
/invite
Sends a link for you to add the bot to your server.
/lower
Sends the message after the command in lower case.
/online
Sends how long the bot has been online.
/ping
Sends pong and latency.
/quote
Sends a random quote from Morgan Pritchard.
/roll
Sends a random number from 1 to a given value.
/say
Sends back the message after the command and deletes your message.
/schedule
Sends a message to a given channel at a given time.
/site
Sends a link to my cool site.
/subscribe
Subscribes to daily messages.
/tictactoe
Play a game of tictactoe with another user, or against a bot.
/title
Sends back the message after the command in title case.
/unsubscribe
Unsubscribes from daily messages.
/upper
Sends back the message after the command in upper case.

Admin Commands

These are the commands only usable from my Discord account.
/broadcast
Sends a message out to all subscribers.
/force_subscriptions
Force send the daily message to all subscribers.
/view_logs
Sends recent bot logs to DMs.
